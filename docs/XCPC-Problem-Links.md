## VJudge / Codeforces 补充说明

- VJudge 上同一套题经常存在多个重现赛或 workbook；上表优先放标题、题源和题目列表最匹配的一条。
- CF Gym 只补充能确认标题或题目列表对应的场次；未确认的行没有强行加入链接。
- 部分 2024/2025 场次目前仅在 QOJ 或 VJudge 可见；若后续 CF Gym 放出，可按同一格式补充。
- QOJ 直达 contest 页面未登录时可能跳转登录页，但 contest ID 仍是对应比赛入口。
