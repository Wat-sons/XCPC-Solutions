# 授权与许可说明

## 设计目标

本仓库同时处理三类对象：

1. **作者个人贡献**：代码、题解、总结、注释、整理文本。
2. **仓库汇编成果**：目录结构、索引、编排、汇总说明、仓库级文档。
3. **第三方材料**：题面 PDF、外部题解链接、引用片段、截图等。

## 基本原则

### 1. 作者保留原始著作权

每个提交者对自己提交的原创内容保留著作权。

这意味着：

- 作者仍然是其代码和题解的原始权利人；
- 仓库中的收录不等于作者放弃权利；
- 作者可以在不违反已作出的授权承诺前提下，另行使用自己的原创内容。

### 2. 提交即授予仓库使用权

向仓库提交内容，视为提交者授予 TYUT XCPC 集训队以下权利：

- 存储、复制、整理、索引、展示、重排；
- 在仓库及其镜像、备份、发布版本中分发；
- 为维护仓库所需进行格式转换、补充链接、添加元数据；
- 在不改变原意的情况下对仓库级说明文档进行整合与修订。

上述授权应视为非独占、全球范围、免版税、可用于仓库维护目的的许可。

### 3. 仓库整体分发采用 CC BY-NC-SA 4.0

除非文件本身另有说明，本仓库作为一个编排后的整体，以 CC BY-NC-SA 4.0 方式对外分发。

这里的“仓库整体”主要指：

- 目录编排；
- 仓库级说明文档；
- 索引与注释；
- 汇总性的编辑内容；
- 未另行标注许可的整理性文本。

### 4. 文件级许可优先

如果某个文件显式标注了不同许可，则该文件以文件内标注为准。

例如：

- 某作者明确声明“仅允许仓库内展示，不允许二次分发”；
- 某外部材料附带更严格的许可限制；
- 某段代码来自第三方且保留原始许可证。

此时应优先遵守文件内标注。

## 第三方材料

### 外部题解链接

外部题解链接本身通常只是引用信息，不等同于转载内容。应尽量保留：

- 来源名称；
- 原始链接；
- 访问时间；
- 简短备注（如题解质量、关键思路）。

### 题面 PDF

题面 PDF 的收录应谨慎处理：

- 优先保留比赛主办方允许公开分发的材料；
- 若版权状态不明确，应优先只保留链接，不直接内置全文；
- 若仓库需要保存题面副本，最好确认其可再分发范围。

### 引用与摘录

引用他人内容时，应尽量做到：

- 标明来源；
- 控制摘录长度；
- 不替代原文；
- 不把受限内容改写成可替代性极强的“伪原创”。

## 备注

这份说明是仓库维护规范，不是法律意见。
