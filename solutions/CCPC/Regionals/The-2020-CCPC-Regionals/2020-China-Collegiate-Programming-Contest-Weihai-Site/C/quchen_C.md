## 题意概述

给定一棵带边权的树，树上有三个人。每个人都有一份候选点列表。

第一个人会从第一个列表中等概率随机选择一个点，记为 $u_1$；  
第二个人会从第二个列表中等概率随机选择一个点，记为 $u_2$；  
第三个人会从第三个列表中等概率随机选择一个点，记为 $u_3$。

三个人选好点之后，需要选择树上的某个点 $v$ 作为会合点，使得三个人到 $v$ 的距离和最小，即最小化：

$$
dis(u_1,v)+dis(u_2,v)+dis(u_3,v)
$$

要求输出这个最小距离和的期望值。

## 关键观察

对于树上的任意三个点 $u_1,u_2,u_3$，存在一个唯一的点 $v$，使得三点到 $v$ 的距离和最小。

这个点可以理解为三条路径的交汇点，也就是树上的“三点中位点”。

对于固定的三个点，有如下结论：


$$
\min_{v}\left(\mathrm{dis}(u_1,v)+\mathrm{dis}(u_2,v)+\mathrm{dis}(u_3,v)\right)=
\frac{\mathrm{dis}(u_1,u_2)+\mathrm{dis}(u_1,u_3)+\mathrm{dis}(u_2,u_3)}{2}
$$

原因是，连接三个点的最小连通子树中，每条边都会被两两距离统计两次，而三个人到最优会合点的距离和正好是这棵小树的边权和。

因此，原问题可以转化为求三组点之间两两距离的期望：

$$
E =
\frac{E[dis(u_1,u_2)] + E[dis(u_1,u_3)] + E[dis(u_2,u_3)]}{2}
$$

于是我们只需要解决一个子问题：

给定两组点 $A,B$，分别从两组中等概率随机选择一个点，求二者距离的期望值。

## 算法设计

考虑如何计算两组点之间的距离和。

对于树上的一条边 $e$，假设它的权值为 $w$。如果删去这条边，整棵树会被分成两部分。

如果随机选择的两个点分别位于这条边的两侧，那么它们之间的路径一定经过这条边，此时这条边对距离贡献 $w$。

如果两个点在同一侧，那么路径不会经过这条边，这条边没有贡献。

因此，对于两组点 $A,B$，边 $e$ 的贡献为：

$$
w \times
(cnt_A \times (|B|-cnt_B) + cnt_B \times (|A|-cnt_A))
$$

其中：

- $cnt_A$ 表示边的一侧中属于集合 $A$ 的点数；
- $cnt_B$ 表示边的一侧中属于集合 $B$ 的点数；
- $|A|,|B|$ 分别表示两组点的总数量。

把所有边的贡献加起来，就得到了所有有序点对 $(a,b)$ 的距离总和。

最后除以 $|A|\times |B|$，即可得到期望距离。

本题有三组点，所以分别计算：

- 第一组和第二组的距离总和，记为 `ans12`；
- 第一组和第三组的距离总和，记为 `ans13`；
- 第二组和第三组的距离总和，记为 `ans23`。

最终答案为：

$$
\frac{1}{2}
\left(
\frac{ans12}{|S_1||S_2|}
+
\frac{ans13}{|S_1||S_3|}
+
\frac{ans23}{|S_2||S_3|}
\right)
$$

代码中用 DFS 统计每个子树内分别有多少个来自三组列表的点：

```cpp
for(int i=1;i<=3;i++)
{
    if(st[i].count(u)) sz[u][i]=1;
}

for(auto [v,w]:e[u])
{
    if(v==fa) continue;
    dfs(v,u);
    for(int i=1;i<=3;i++)
    {
        sz[u][i]+=sz[v][i];
    }
}
```
